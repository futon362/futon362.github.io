Discord BBS（掲示板）使い方ガイド

このページでは、ロント兼中央連邦国サーバー内に設置されているBBS（掲示板チャンネル）の使い方を説明します。

BBSは、チャットとは違い「後から読み返しやすい投稿」を目的とした半フォーラム型の投稿スペースです。

BBSの目的

  • 重要な情報を流れずに残す
  • 国家間の連絡や通達を整理して投稿する
  • 長文・議論・提案などを読みやすく管理する
  • 国ごとの告知や制度説明をまとめて掲載する

BBSは、チャット欄のように流れていかないため、“公式掲示板” として扱われます。

投稿ルール

  1. 話題ごとにスレッドを作成してください。
    • 例：国家設立報告、条約提案、イベント告知など
  2. スレッド名は内容が分かるように書く。
    • 例：「中央連邦国：税率改定のお知らせ」
  3. 返信は必ずスレッド内で行う。
    • 本文チャンネルに直接返信しない
  4. 荒らし・煽り・暴言は禁止。
  5. 国家の公式発表は代表者のみ投稿可能。
    • 一般国民はスレッド内で意見を述べる形にする
  6. 議論が長くなる場合は専用スレッドを作成する。

スレッドの作り方

  1. BBSチャンネルを開く
  2. メッセージ入力欄の右にある ＋ を押す
  3. 「スレッドを作成」を選択
  4. タイトルを入力
  5. 必要なら最初の投稿を添えて作成

投稿の種類（推奨分類）

1. 国家関連

  • 国家設立報告
  • 国旗・国名の変更
  • 法律（ローカル法）の追加
  • 国家イベントの告知

2. 国際関係

  • 条約提案
  • 国境問題
  • 紛争の申し立て
  • 仲裁依頼

3. サーバー全体

  • 運営からのお知らせ
  • ルール改定
  • イベント開催情報

4. プレイヤー提案

  • 新システムの提案
  • 経済改善案
  • 建築・交通インフラの提案

5. その他

  • 雑談スレッド（公式発表以外の話題）

通知の扱い

  • BBSは重要情報が多いため、**通知ON推奨**
  • スレッドごとに通知設定を変更できます
  • 不要なスレッドはミュートしてOK

モデレーション（管理）

以下の行為は削除対象です：

  • 無関係な話題
  • 荒らし・煽り
  • スパム
  • 他国への誹謗中傷
  • ルール違反の投稿

📄 まとめ

  • BBSは「流れない情報」を残す場所
  • スレッド単位で話題を整理
  • 国の公式発表はここに集約
  • 国際関係の議論もここで行う
  • 読み返しやすく、管理しやすいのが特徴
